## [0.0.1] Foundation Update

This is the first public development build of OrcSlime.  
It introduces the basic game structure and the first playable land plot.

### Added

- A working in-game time system
- Pause and resume functionality
- A structured game foundation for future updates
- A 3x3 land plot
- Two starting trees
- One chest
- Adjustable tile size
- Adjustable spacing between tiles
- Adjustable outer spacing around the plot
- Automatic resizing of the game area based on plot size

### Improvements

- Cleaner and more scalable game structure
- Plot layout is fully configurable
- Game systems are prepared for future features such as interaction, harvesting, and expansion

### Notes

This update focuses on building a strong foundation for the game.  
Future updates will introduce interaction, tree growth, harvesting mechanics, and additional gameplay systems.
